🔥 مميزات البرنامج

إدارة العملاء

تسجيل العملاء الجدد والقدامى، متابعة الأقساط، الصيانة، والتحكم بأنواع العملاء (كاش، أقساط، مجاني، خارجي).

إدارة المخزن

تسجيل المنتجات والأجهزة مع الكمية وسعر الشراء وسعر البيع، حساب الربح الصافي، وإضافة صور الموردين.

المبيعات

تسجيل عمليات البيع بالتاريخ والوقت مع حساب صافي الربح وطباعـة التقارير الشهرية والأرشيف.

إدارة الصيانة

متابعة الصيانة الدورية للأجهزة والفلاتر، تجديد الشمعات، تسجيل المهام اليومية والمتأخرة والتقارير.

إدارة الرواتب

إدارة رواتب الموظفين مع تسجيل السلف، الخصومات، المكافآت، وأرشيف الرواتب السابق.

إدارة المستخدمين

إضافة مستخدمين جدد وتحديد صلاحياتهم (مدير، موظف، إدخال بيانات) مع تحكم كامل في الأذونات.

الإحصائيات والتقارير

عرض شامل للأرباح، المصروفات، الديون، قيمة المخزون، العملاء النشطين، والفنيين الأكثر نشاطًا.

عملاء الإعلان

إدارة قاعدة بيانات العملاء المحتملين (Prospects) مع أدوات بحث وتصفية وحفظ نسخة احتياطية.

إدارة المستهلكات

تسجيل المصروفات والمستهلكات المختلفة، خصمها من الأرباح، وحساب صافي الربح بدقة.

خدمة العملاء

صفحة دردشة مدمجة مع العملاء لمتابعة المشاكل والطلبات بشكل لحظي وسريع.

السيرة الذاتية للموظفين

إضافة وتحديث بيانات العاملين مع صور شخصية أو بطاقة قومية وخيارات تعديل/حذف/بحث.

دليل المستخدم

شرح كامل للبرنامج، مواصفات الأجهزة، وطريقة الاستخدام خطوة بخطوة.
