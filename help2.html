🔐 الأمان والخصوصية

  • تخزين محلي فقط: بيانات النظام تُخزن محليًا على جهاز العميل في ملف قاعدة بيانات SQLite يتم إنشاؤه تلقائيًا عند التثبيت. لا يتم إرسال بيانات العملاء أو سجلات المبيعات إلى خوادم خارجية ما لم يُفعل المستخدم ذلك صراحة (مثل النسخ الاحتياطي أو تصدير CSV).
  • التحكم الكامل: صاحب النظام يملك تحكم كامل بملفات البيانات ويمكنه عمل نسخ احتياطية أو حذفها يدويًا.
  • التشفير والنسخ المحلية: يُنصح باستخدام تشفير القرص أو حلول النسخ الاحتياطي المشفرة. يمكن إضافة طبقات تشفير للتطبيق عند الطلب (خدمة مدفوعة).
  • صلاحيات الوصول: صفحات حساسة مثل "إضافة مستخدم" و"إدارة المستخدمين" مرئية فقط لحساب المدير/المسؤول.

🛠️ الدعم الفني والصيانة

  • دعم مباشر: عبر برنامج XMail WPF (C#) — فتح تذكرة، إرسال لوجات، وجدولة جلسة مشاركة شاشة.
  • الاستجابة: دعم للحالات الحرجة بنظام الأولوية. التحديثات وتصحيح الأعطال تتم عبر حزم تحديث صغيرة.
  • تدريب ومساعدة: دليل مستخدم مفصل + جلسة تدريبية أولية عند الطلب (قد تكون مدفوعة).

⚡ الأداء والسعة

  • بلا حدود عددية: لا يوجد حد لعدد المستخدمين أو العملاء.
  • اختبارات عملية: تمت التجربة على قاعدة بيانات فيها 20,000 سجل عميل، وزمن البحث ~3 ثوانٍ.
  • نصائح للأداء: استخدام SSD ورفع حجم الذاكرة يحسن الأداء. يُوصى بتنظيف البيانات وأرشفة شهرية.

💸 التجربة المجانية وسياسة الاسترجاع

  • الفترة التجريبية: تجربة مجانية لمدة 3 أيام تشمل جميع الوظائف.
  • استرجاع كامل خلال 14 يومًا فقط في حال:
    • وجود خلل فني جوهري يمنع الاستخدام (مثل فشل كامل في فتح قاعدة البيانات أو فقدان بيانات بسبب خطأ في البرنامج).
    • تعذر إصلاح العطل خلال 7 أيام عمل بعد استلام تفاصيل المشكلة.
  • لا يتم الاسترداد في الحالات التالية:
    • مشكلات إعدادات الجهاز (نقص صلاحيات، قرص ممتلئ، برامج حماية تمنع عمل التطبيق).
    • سوء استعمال أو حذف بيانات يدويًا.
    • طلب بعد 14 يومًا من الشراء (إلا باتفاق خاص).
  • إثبات الخطأ: يجب تقديم لوجات التطبيق (log files) ووصف للمشكلة. يتم الرد خلال 72 ساعة عمل.
  • استرداد جزئي أو إصلاح بديل: قد يُعرض حل بديل أو تعويض جزئي إذا أمكن.
  • اتفاقيات خاصة: يمكن توقيع اتفاقيات دعم (SLA) تحتوي شروط استرداد وتعويض مخصصة.

💡 نصيحة: إضافة صفحة داخل التطبيق لتصدير اللوجات تلقائيًا يسهل الدعم ويُسرع الحلول.
